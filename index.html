Produto 1

JavaScript é uma linguagem de programação usada principalmente para tornar páginas da web interativas. Pense nela como a parte "dinâmica" da internet. Enquanto o HTML define a estrutura de uma página (o texto, as imagens, os links) e o CSS a deixa bonita (as cores, o layout, as fontes), o JavaScript é o que dá vida à página.

Produto 2

HTML, que significa HyperText Markup Language (Linguagem de Marcação de Hipertexto), é a espinha dorsal da web. Ele não é uma linguagem de programação, mas sim uma linguagem de marcação. Imagine que você está construindo uma casa. O HTML seria o esqueleto: as paredes, as fundações, o telhado. Ele define a estrutura e o conteúdo de uma página da web.
